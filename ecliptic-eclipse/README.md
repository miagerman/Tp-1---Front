Catálogo de Comidas

Explicación del proyecto

Este proyecto es una página web que hice usando Astro.  
La página muestra un catálogo de algunas de mis comidas favoritas.  
Las comidas aparecen en tarjetas con su nombre, categoría y una descripción.

También se puede buscar comidas escribiendo en el buscador o filtrarlas por categoría usando el selector.

La página tiene varias secciones como catálogo, sobre el proyecto y contacto.

Herramientas usadas

Para hacer este proyecto usé:

- Astro
- HTML
- CSS
- JavaScript
- Node.js
- Visual Studio Code

Instrucciones para ejecutarlo

1. Abrir la carpeta del proyecto en Visual Studio Code.

2. Instalar las dependencias escribiendo en la terminal:

npm install

3. Ejecutar el proyecto con:

npm run dev

4. Abrir el navegador y entrar a:

http://localhost:4321
